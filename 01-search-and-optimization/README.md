# Search and Optimization

This folder contains cleaned portfolio versions of my implementations for classical AI search and optimization topics:

- A* search and DFS on a custom state-space planning problem.
- Simulated annealing for a knapsack-style combinatorial optimization task.

Key concepts: state-space modeling, graph construction, heuristic search, admissible heuristics, local search, simulated annealing, and randomized optimization.
