# Course Summary

**Course:** CE417 Artificial Intelligence  
**Institution:** Sharif University of Technology  
**Term:** Spring 2023  

## High-Level Description

The course introduced artificial intelligence from theoretical and practical perspectives, with emphasis on designing intelligent agents, state-space search, knowledge representation and inference, and machine learning.

## Main Topics Reflected in This Repository

1. Search and optimization
2. Constraint satisfaction
3. Adversarial search
4. Reinforcement learning
5. Bayesian networks and probabilistic inference
6. Classical machine-learning models
